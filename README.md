# testing-templates
Testing implementations in 

- Node.js (Chai and Mocha)

    Command  : <b>npm run tests</b>

- React.js (Jest)

    Command  : <b>npm run tests</b>
    
- Golang 

    Command  : <b>go test</b>
    
- Python (unittest)

    Command  : <b>python testpython.py</b>
    
- Java  (JUnit)

- Solidity (Chai - Smart Contract Testing with Truffle)

    Command  : <b>truffle test</b>
